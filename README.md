# Heart Disease Prediction App

A machine learning web application that predicts whether a person is at risk of heart disease based on medical parameters.

## Project Overview

Heart disease is one of the leading causes of death worldwide. This project uses a trained machine learning model to predict the possibility of heart disease using patient details such as age, cholesterol, chest pain type, blood pressure, ECG results, and more.

The application is built using Python and Streamlit.

## Features

* Predicts heart disease risk
* User-friendly interface
* Attractive Streamlit design
* Uses trained machine learning model
* Fast and simple predictions
* Can be deployed online

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Joblib

## Input Features

The model uses the following medical features:

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG
* Maximum Heart Rate
* Exercise Induced Angina
* Oldpeak
* Slope
* Number of Major Vessels
* Thalassemia

## Prediction Output

* High Risk of Heart Disease
* Low Risk of Heart Disease
